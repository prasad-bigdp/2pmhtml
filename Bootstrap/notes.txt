Bootstrap is CSS component library which is used for responsive web design. A library is a pre written code for specific purpose
How to connect Bootstrap to HTML?
i. connect Bootstrap cdn link to html using link tag

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.6/dist/css/bootstrap.min.css" rel="stylesheet">

 ii. apply class names as per bootstrap offical website "getbootstrap".

 Most widely used class names are:
  i. p --> padding
     p-0 --> no padding, p-1 -->0.5em padding, p-2 -->1 em padding......till p-5
     px- padding x axis(left and right),py - padding y axis(top and bottom), pt- padding top,pb- padding bottom, ps-padding start(left), pe -- padding end(right)
  ii. m - margin
  iii. h-height,w-width
  iv. d-- display( d-block, d-inline, d-flex etc..,)
  iv. float-start,float-end,float-none

  Bootstrap Provides 8 colors such as 
  primary(blue), secondary(gray), success(green), danger(red), warning(yellow), info(skyblue), dark(black), light(white)
